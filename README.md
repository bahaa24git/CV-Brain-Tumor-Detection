# Brain Tumor Detection from MRI Images

This repository contains a computer vision project for detecting brain tumors from MRI images using image-processing and machine-learning workflows.

## Project Goal
Train and evaluate a pipeline that can classify MRI scans into tumor-related categories using computer vision techniques.

## Notebook
- `cv-project-v1-3-0.ipynb` — main project notebook

## Dataset
The notebook is designed for the Brain Tumor MRI dataset, with training and testing folders.

## Repository Notes
- The dataset is intentionally excluded from GitHub to keep the repository lightweight.
- The notebook includes preprocessing, feature extraction, segmentation, and classification experiments.

## How to Use
1. Open the notebook in Jupyter.
2. Ensure the required Python libraries are installed.
3. Point the notebook to your local dataset folder.

## Tech Stack
- Python
- OpenCV
- scikit-image
- scikit-learn
